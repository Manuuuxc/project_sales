# Screenshots

Adicione aqui os prints reais do dashboard para que eles possam aparecer na documentacao do GitHub.

Nomes recomendados:

- `visao-executiva.png`
- `receitas.png`
- `despesas.png`
- `resultado-financeiro.png`
- `detalhamento.png`

Para gerar os prints:

1. Abra `dash_financeiro.pbip` no Power BI Desktop.
2. Acesse a pagina desejada do relatorio.
3. Tire o print em 16:9, de preferencia com a janela maximizada.
4. Salve a imagem nesta pasta usando um dos nomes recomendados acima.
